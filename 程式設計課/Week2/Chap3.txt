一般常用的資料型態有以下幾種:
int (整數)
short (一半大小)
long (兩倍大小)
long long
float
double (精度是float兩倍)
long double
char
enum (列舉)
void (空) -> 在宣告函式時 回傳型態寫void表不回傳 參數中寫void表不傳參數

宣告時可使用unsigned關鍵字告知編譯器不需要signed bit，可讓變數的可儲存數字上限x2；使用const宣告變數可保護變數不被修改

函式名稱只能使用英文字母、數字、底線符號，不可為數字開頭，且不可使用保留字

算術運算子:
+ - * / % (取餘數)

邏輯運算子:
&& || (OR) ! (NOT)

位元運算子:
& | (OR) ^ (XOR) >> (右移 相當於/2) << (左移 相當於*2) ~ (NOT)

複合運算子:
i = i + a; 可縮寫成 i += a;

遞增遞減預算子:
i++ == i += 1
i-- == i -= 1
++放後面 -> 先取出 再遞增
++放前面 -> 先遞增 再取出

條件(三元)運算子:
(條件) ? True時回傳資料 : False時回傳資料
Ex: a == 0 ? 1 : 0;

型態轉換:
顯式型態轉換: (型態名稱)變數
atoi 字串轉整數
atof 字串轉浮點數
atol 字串轉long
itoa 整數轉字串
fcvt 浮點數轉字串
